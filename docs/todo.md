## 다음 작업: 공용 사이트 시스템

현재 단계에서의 공용 시스템은 게임 내부 기능이 아니라,
사이트 전체의 진입 구조를 먼저 구성하는 것을 의미한다.

### 목표

- 루트 페이지는 게임 화면이 아니라 로그인과 게임 목록을 제공하는 런처 페이지로 사용한다.
- 사용자는 첫 페이지에서 로그인 상태를 확인하고, 원하는 게임을 선택해 진입한다.
- 각 게임은 `games/` 내부에서 독립적인 `index.html`을 가진다.
- Canvas, 게임 루프, 입력 처리 등은 개별 게임 내부 구현 단계에서 다룬다.
