## 현재 개발 우선순위

현재 우선순위는 게임 내부 시스템이 아니라 사이트 공용 진입 구조를 만드는 것이다.

중요한 기준:

- 루트 `index.html`은 게임이 아니다.
- 루트 페이지는 로그인 버튼과 게임 목록을 가진 런처 페이지다.
- Canvas, 게임 루프, 입력 처리, 게임 UI는 아직 공용 시스템으로 고려하지 않는다.
- 게임 관련 구현은 각 게임 폴더 내부에서 진행한다.
- 공용 코드는 `shared/`에 둔다.
- 먼저 구현할 흐름은 `접속 → 로그인 → 게임 선택 → 게임 실행`이다.
